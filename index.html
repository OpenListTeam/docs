    OpenList

    🗂️ A file list program that supports multiple storage, powered by Gin and Solidjs.

    Get started💡GitHub🌱Privacy PolicyTerms of Use

    Easy to Use

    OpenList was designed from the ground up to be easily installed and it can be used on all platforms.

    Multiple storage

    OpenList supports multiple storage providers, including local storage, Aliyundrive, Onedrive, Google Drive, etc., and is easily expand.

    Support WebDAV

    OpenList supports all storage WebDAV, which is a standard for accessing files on.

    Dark Mode

    Switch between light and dark modes freely

    Protected routes

    Add password protection and authentication for specific path

    File preview

    Support video, audio, office, pdf, code, image preview, etc. Even ipa install

    Package download / Batch download

    Use stream api of brower to support package download without server usage / Batch download support folder using Aria2

    single sign-on

    Use single sign-on to quickly log in to OpenList

    Automatic registration of OpenList account

    Use single sign-on to automatically register as an OpenList account to quickly register

    offline download

    To download the torrent content offline to the specified directory, a good network environment is required

    Crypt encrypt/decrypt files

    Anyone can securely store encrypted data on remote storage providers. Data is stored in a Crypt, and the provider can only see the Crypt, not your data.

    More new features

    Including text editor, readme/html render, file permalink, cloudflare workers proxy, etc.
